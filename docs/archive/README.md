# Archivo de documentación

Este directorio conserva contexto histórico que ya no guía el trabajo cotidiano. No es una fuente de verdad para implementar cambios nuevos.

- [`csv/`](./csv/) — configuración inicial, diagramas y cambios históricos de la importación CSV.
- [`implementation/`](./implementation/) — planes y resúmenes de funcionalidades ya implementadas.
- [`monitoring/`](./monitoring/) — auditorías y planes reemplazados del módulo de monitoreo.
- [`incidents/`](./incidents/) — diarios técnicos e informes de incidentes resueltos.

Antes de reutilizar un documento archivado, compáralo con el código, las migraciones y las referencias enlazadas desde [`../README.md`](../README.md).
